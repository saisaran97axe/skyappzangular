HOW IOT WORKS

SENSOR TECHNOLOGY

Sensors can be in the form of motion sensors, door locks or light bulbs etc, which are implanted with the devices and used to sense the weather/environment conditions, track the location and other relevant purposes

IOT GATEWAYS

IoT gateway is a form of solution that connects the devices to the internet through the mode of communication like RFID, Bluetooth, wifi, etc. This set-up also allows the devices to connect with each other.

CLOUD/SERVER

Data that needs detailed processing and where feedback is delayed, gets forwarded to cloud based systems, where more sophisticated IT Systems can analyze, manage and safely store the data. This processed data then perform intelligent actions.

MOBILE APP

Intuitive Mobile apps will help end users to monitor and send commands to their smart devices from any location and the user will be able to affect the system in the desired way by performing an action.
